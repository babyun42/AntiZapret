Tor Bridges (Мосты)

Если стандартный internet сильно заглушен, Tor с мостами поможет восстановить доступ к сети. Доступен на любые платформы.

Android
Видео гайд Официальный сайт Конфиги
Google Play F-Droid
1. Скопируйте список мостов (конфигов) с сайта.
2. Установите приложение и выберите опцию «Подключиться напрямую к Tor».
3. Зайдите в раздел настройки мостов, выберите «Добавить новые мосты» и вставьте скопированные данные.
4. Если кнопка сохранения не нажимается, поищите мосты с восклицательным знаком.
5. Активируйте использование мостов. В настройках приложений рекомендуем временно отключить российские сервисы. Нажмите «Подключить» и дождитесь завершения.
Windows
Video гайд Официальный сайт GitHub Конфиги
1. Скачайте onionFruit через релизы в гитхаб или через офиц. сайт.
2. Копируйте конфиг для onionFruit.
3. Нажмите настройки.
4. Нажмите мосты.
5. Измените disabled на plain, ну или которая вам нужна.
6. Вставьте мосты и нажмите кнопку добавить.
7. Закройте вкладку.
8. Нажмите на переключатель и ждите.

Amnezia VPN / WG

Отличный вариант для изменения вашего местоположения. Использует модифицированый протокол WireGuard.

Android
Видео гайд Официальный сайт Сайт для конфига
1. Скачайте APK-файл из релизов на GitHub (выбирайте версию, подходящую под ваше устройство).
2. Перейдите на сайт генерации конфигов, выберите рабочее зеркало и нажмите «Сгенерировать конфиг».
3. Пройдите капчу и скачайте файл в формате .conf.
4. В приложении Amnezia VPN нажмите «Приступим» -> «Файл с настройками подключения».
5. Выберите скачанный файл (warp...conf) и нажмите «Подключить». Разрешите VPN-запрос в системе.
Windows
Видео гайд Официальный сайт Сайт для конфига
1. Перейдите в релизы на GitHub разработчиков и скачайте файл с расширением .exe.
2. Откройте сайт для создания конфигов, выберите рабочее зеркало (например, основное) и нажмите «Сгенерировать зеркало».
3. Пройдите капчу и нажмите кнопку «Скачать конфиг».
4. В приложении Amnezia VPN нажмите «Приступим» -> «Файл с настройками подключения».
5. Через открывшийся проводник найдите и выберите скачанный файл (warp...conf), чтобы автоматически импортировать настройки , всё заработало.

Утилиты для обхода

Zapret Discord YouTube

Утилита для модификации трафика на лету на Windows. Отлично подходит для восстановления работы голосовых чатов Discord и быстрой загрузки видео на YouTube.

Windows
Видео гайд GitHub
1. Скачайте архив (.rar) из релизов разработчика и распакуйте его.
2. Найдите в папке файл «general (ALT11).bat» and запустите его (желательно от имени администратора).
3. Окно консоли должно оставаться открытым. Теперь Discord и YouTube будут работать стабильно.

TG WS Proxy (Telegram)

Если Telegram долго грузит медиа или не подключается, используй WebSocket прокси.

Android
Видео гайд GitHub
1. Установите версию 1.3.0 (.apk) с GitHub. Обязательно разрешите приложению работу в фоне.
2. В настройках оставьте стандартный порт, установите «Пул WS» на значение 4 и включите автозапуск.
3. Нажмите на значок Telegram в приложении и выберите «Применить в Telegram».
4. В открывшемся мессенджере нажмите «Подключить прокси».

Примечание: иногда через этот метод может не работать отправка тяжелых файлов.
Windows
Видео гайд GitHub
1. Скачайте TG WS Proxy через релизы на GitHub, скачивайте с окончанием .exe.
2. Заходите в трее ищите TG WS Proxy, нажимайте по нему правой кнопкой мыши, настройки, листайте в низ, нажимайте на автозапуск, сохранить.
3. Заходите опять в трее, нажимайте правой кнопкой мыши по TG WS Proxy и выберайте "Открыть в Telegram" или "Скопировать ссылку".
4. Вас перекидывает в Telegram, нажимайте "Подключить прокси" и все хорошо работает.

Умные маршруты

Что такое белые списки?

Вместо того чтобы пускать ВЕСЬ ваш трафик через зарубежный сервер, вы можете использовать белые списки. ВПН будет включаться автоматически только для заблокированных сайтов.

Как установить?

  1. Скачайте и установите совместимый клиент.
  2. Загрузите файл конфигурации правил маршрутизации.
  3. В настройках выберите режим "Обход LAN и России".
